# PowerDNS-Admin
PowerDNS Server Web Administrator in Angular 16 with Microsoft .net 8 back-end.

Project was moved to: [Azure DevOps](https://dev.azure.com/AdamDebowski/PowerDNS-Admin)

# Bakck-end
Technology stack:
- C#
- .NET 8 (LTS)
- Entity Framework 8
- Serilog (Sinks: file)
- SignalR
- SQLite (so far)

# Front-end
Technology stack:
- Angular 16 (so far)
- SCSS
- Angular Material 16 (so far)
- RxJS
- Signal R Client
